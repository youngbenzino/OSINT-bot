# OSINT-bot

**OSINT-bot** — Python-бот для сбора и анализа публичной информации из Telegram-каналов.  
Инструмент предназначен для исследования публичных сообществ: собирает посты и комментарии, извлекает авторов, считает статистику и генерирует удобный отчет (топ комментаторов, ссылки на комментарии, базовая информация об аккаунтах). Подходит для учебных и исследовательских задач. Не использовать для нарушения приватности или законов.

---

## Ключевые возможности

- Анализ канала — выборочный или «всё подряд» (без ограничений при аккуратной настройке).  
- Сбор комментариев и сопоставление авторов (username / id / имя).  
- Топы комментаторов, метрики (комментариев, скорость сбора, время анализа).  
- Просмотр комментариев с прямыми ссылками и постраничная навигация.  
- Анализ аккаунта: базовая и расширенная информация (через Telethon GetFullUser/GetFullChannel).  
- Адаптивные задержки и простая защита от флуда (настроиваемые параметры).

---

## Как это работает (коротко)

1. Бот и пользовательский Telethon-клиент (два клиента: `bot_client` — для взаимодействия в Telegram, `user_client` — для запросов API) запускаются из `main.py`.  
2. Пользователь через интерфейс бота выбирает режим: анализ канала или аккаунта.  
3. Для канала бот получает сообщения (через `iter_messages`), собирает ответы/реплаи, обрабатывает авторов и накапливает статистику в памяти.  
4. Для аккаунта выполняются запросы `GetFullUserRequest` или `GetFullChannelRequest` для расширенной информации.  
5. Результаты выводятся в Telegram: сводная статистика и интерактивные кнопки для просмотра деталей.

Внутренние сущности: `UserData` (состояние сессии пользователя), кеш найденных авторов (`found_users`), прогресс- и статус-сообщения.

## Требования

- Python 3.10+  
- Библиотеки: `telethon` (и другие зависимости, если добавлены)  
- Активный Telegram API (api_id + api_hash) и bot token (BotFather)
